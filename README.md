# 🎵 Task 1 – Music App Authentication UI

Designed a modern and user-friendly authentication interface for a Music Application, including a splash screen, login page, and sign-up page.

### Features

* Splash Screen
* Login Screen
* Sign Up Screen
* Email & Password Fields
* Password Visibility Toggle
* Social Login Options (Google, Facebook & Apple)
* Clean and Minimal Interface
